## Remix

Remix est un framework bâti sur React depuis 2022, par la même équipe que React Router. Financé par Shopify.

Autres frameworks bâtis sur React:

- Next JS
- Create-react-app (CRA)
- Gatsby
- T3
- Vite

---

### React vs Next JS

Next JS est plus populaire que Remix. (Surtout parce que Remix est très récent.)

En octobre 2023, la popularité de Remix n'est que 0.5% de la popularité de Next JS.

Article qui décrit les différences entre React et Remix:

https://prateeksurana.me/blog/nextjs-13-vs-remix-an-in-depth-case-study/

---

### Caractéristiques de Remix

- Framework complet
- Encourage l'usage de html/js standardisé: `Request`, `Response`, `FormData`, etc.
- Permet un layout imbriqué
- Chargement de données en parallèle data in parallel
- Gestion des situations de concurrence (race conditions)
- Permet d'afficher le HTML avant le chargement du JS

- Ne supporte pas encore de **_Server-Side Rendering_** (SSR) ou **_React Server Components_** (RSC), (prévu pour le futur)
- Permet une forme de **_Static Site Generation_** (SSG) optimizé pour le dynamique

### Démarrer avec Remix

```bash
npx create-remix@latest --install --init-script --git-init --template epicweb-dev/epic-stack
```

```bash
mkdir my-remix-app
cd my-remix-app

# install runtime dependencies
npm i @remix-run/node @remix-run/react @remix-run/serve isbot react react-dom

# install dev dependencies
npm i -D @remix-run/dev

```
